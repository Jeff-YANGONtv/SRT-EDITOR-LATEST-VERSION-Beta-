## 🔹Overview
**YGN TV Subtitle Editor** သည် မြန်မာစာတန်းထိုး (Burmese Subtitles) များအား ပိုမိုမြန်ဆန်၊ သပ်ရပ်စွာ တည်းဖြတ်နိုင်ရန် Rules လုပ်ငန်းသုံး Tool တစ်ခုဖြစ်သည်။ Browser ပေါ်တွင် တိုက်ရိုက်အသုံးပြုနိုင်ပြီး Telegram Bot များ၊ Google Sheets များနှင့် ချိတ်ဆက်ထားသဖြင့် လုပ်ငန်းခွင် စီမံခန့်ခွဲမှုကို ပိုမိုကောင်းမွန်စေပါသည်။

## 🗝️ Key Features

* **Premium Glassmorphism UI:** အမြင်အေးပြီး ခေတ်မီသော Dark-mode UI interface။
* **SRT Parsing & Sync:** SRT ဖိုင်များကို Error ကင်းစွာ ဖတ်ရှုပြင်ဆင်နိုင်ခြင်း။

* **🎞️16:9 Reference Player:** ဘာသာပြန်စာသားများကို Video နှင့် တိုက်စစ်ရန် Local Video Player ပါဝင်ခြင်း။

* 🌐Instant Server Sync:** ပြင်ဆင်ပြီးသော ဖိုင်များကို Telegram Channel များသို့ တိုက်ရိုက်ပေးပို့ခြင်းနှင့် Google Sheets တွင် Log မှတ်ခြင်း။

* 🔎Smart Toolbar:** Search, Undo (အမှားပြင်ဆင်ချက်ပေါင်း ၃၀ အထိ) နှင့် Auto-Broom (စာကြောင်းအလွတ်များ ရှင်းလင်းခြင်း)။

* 📱Responsive Design:** မည်သည့် Mobile Phone ဖြင့်မဆို အဆင်ပြေစွာ သုံးနိုင်ခြင်း။


## 🛠️ Technical Stack

- **Frontend:** React.js, Next.js, Framer Motion

- **Styling:** Tailwind CSS (Custom Glassmorphism)

- **Backend Sync:** Telegram Bot API & Google Apps Script
